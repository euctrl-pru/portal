--- title: Turnaround ---

Turnaround shows the time elapsed between an aircraft blocking in and blocking off for the next flight from the same parking stand on the same day. The sample is treated to consider only ground times with the following characteristics

Seats WTC Cutoff time1
M H J
B 31-80 x 90 minutes
C 81-180 x x 120 minutes
D 181-220 x x 150 minutes
E > 220 x x x 180 minutes
1 Turnaround cutoff time based on scheduled Ground Time.

Additional information, based on Network Manager data, can be found in the tool MIRROR at https://www.eurocontrol.int/tool/mirror
