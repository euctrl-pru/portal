Data source

The data is supplied by the EUROCONTROL Aviation Sustainability Unit.

Data description

The data provides total CO2 emissions by State based on departures.

Column naming and types

FLIGHT_MONTH
YEAR Year
STATE_NAME State Name
STATE_CODE State Code
Y_CO2_QTY_TONNES Co2 Qty for the entire year (for that state)
Y_TF traffic for the entire year (for that state)
